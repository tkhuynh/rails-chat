# Rails Live Chat

Live Chat App created with Rails 5, Chanel and ActionCable

## How to Use?
* Clone the repo
* ```bundle install```
* ```rake db:create``` Make sure you have PostgreSQL sever running
* ```rake db:seed```
* ```open http://localhost:3000```
